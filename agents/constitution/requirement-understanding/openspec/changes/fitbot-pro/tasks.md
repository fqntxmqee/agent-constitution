## 阶段 1: 项目初始化

### Task-001: 创建项目结构

**描述:** 初始化 FitBot Pro 项目目录结构

**执行步骤:**
```bash
# 1. 创建项目目录
mkdir -p fitbot-pro/src/{config,services,models,repositories,utils,types}
mkdir -p fitbot-pro/data/imports
mkdir -p fitbot-pro/tests/{services,models,integration}
mkdir -p fitbot-pro/docs

# 2. 初始化 npm 项目
cd fitbot-pro
npm init -y

# 3. 安装依赖
npm install typescript @types/node
npm install better-sqlite3
npm install axios
npm install dayjs
npm install node-cron

# 4. 配置 TypeScript
npx tsc --init
```

**验收标准:**
- [ ] 目录结构完整
- [ ] package.json 配置正确
- [ ] tsconfig.json 配置正确

**预计工时:** 1h
