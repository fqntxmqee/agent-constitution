# FitBot Pro 健身教练智能体 - 项目提案

## 1. 项目概述

### 1.1 项目名称
**FitBot Pro 健身教练智能体 (专业版)**

### 1.2 项目代号
`fitbot-pro`

### 1.3 优先级
**P1** - 高优先级个人应用

### 1.4 项目类型
标准构建流 - 开发类任务（个人健身智能体）

---

## 2. 用户背景分析

### 2.1 用户画像

| 属性 | 详情 |
|------|------|
| **健身经验** | 1 年 |
| **健身水平** | 中级 |
| **运动类型** | 哑铃三分化力量 + 骑车 + 跑步 |
| **健身场所** | 家庭 + 户外 |
| **数据来源** | 华为手表 |

### 2.2 运动能力基线

| 项目 | 水平 | 目标 |
|------|------|------|
| 跑步 (半马) | 2:10 | 2:00 |
| 骑车 (日常) | 1 小时/天 | 维持 |
| 力量训练 | 三分化 | 增肌 + 力量提升 |

---

## 3. 功能需求

### 3.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 运动规划 | 三分化力量 + 跑步 + 骑车安排 | P0 |
| 饮食规划 | 高蛋白 + 训练日/休息日差异 + 跑步营养 | P0 |
| 数据分析 | 华为手表数据集成 | P0 |
| 进度追踪 | 力量/耐力/体重/体脂 | P0 |
| 提醒功能 | 训练/饮食/恢复提醒 | P1 |
| 飞书集成 | 机器人消息 + 周报推送 | P1 |

### 3.2 运动规划详情

#### 三分化力量训练

| 训练日 | 肌群 | 动作 |
|--------|------|------|
| Day 1 | 推 (胸/肩/三头) | 哑铃卧推、肩推、臂屈伸 |
| Day 2 | 拉 (背/二头) | 哑铃划船、引体向上、弯举 |
| Day 3 | 腿 (股四头/腘绳/小腿) | 深蹲、硬拉、弓步蹲 |

#### 有氧安排

| 运动 | 频率 | 时长 | 强度 |
|------|------|------|------|
| 骑车 | 每天 | 1 小时 | 中低强度 |
| 跑步 | 2-3 次/周 | 30-60 分钟 | 间歇/耐力 |

---

## 4. 技术选型

| 组件 | 技术选择 | 理由 |
|------|----------|------|
| 运行时 | Node.js | 与 OpenClaw 一致 |
| 前端 | 飞书小程序 | 用户已使用飞书 |
| 数据存储 | SQLite + JSON | 轻量、本地存储 |
| 华为数据 | 手动导入 + API | 兼容多种数据源 |
| AI 建议 | 规则引擎 + LLM | 个性化建议 |
| 消息推送 | 飞书机器人 | 用户指定 |

---

## 5. 项目范围

### 5.1 包含 (In Scope)

- 三分化力量训练计划生成
- 跑步训练计划 (半马提升)
- 骑车训练安排
- 饮食规划 (高蛋白)
- 华为手表数据导入/同步
- 训练负荷/心率/睡眠/恢复分析
- 进度追踪与可视化
- 飞书机器人消息推送
- 周报自动生成

### 5.2 不包含 (Out of Scope)

- 实时心率监控
- 视频动作指导
- 社交功能
- 电商/商城集成
