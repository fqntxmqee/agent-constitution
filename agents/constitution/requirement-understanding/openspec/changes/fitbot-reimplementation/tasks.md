## 阶段 1: 项目初始化

### Task-001: 创建智能体目录结构

**描述:** 创建 FitBot 智能体目录结构

**执行步骤:**
```bash
# 1. 创建智能体目录
mkdir -p agents/fitbot/src/{services,models,repositories,utils,types}
mkdir -p agents/fitbot/data/imports
mkdir -p agents/fitbot/tests/{services,models,e2e}
mkdir -p agents/fitbot/docs

# 2. 初始化 npm 项目
cd agents/fitbot
npm init -y

# 3. 安装依赖
npm install typescript @types/node
npm install better-sqlite3
npm install dayjs
npm install csv-parse

# 4. 配置 TypeScript
npx tsc --init
```

**验收标准:**
- [ ] 目录结构完整
- [ ] package.json 配置正确
- [ ] tsconfig.json 配置正确

**预计工时:** 2h
