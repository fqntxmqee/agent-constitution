## 3. 用户背景分析

### 3.1 用户画像

| 属性 | 详情 |
|------|------|
| **健身经验** | 1 年 |
| **健身水平** | 中级 |

|------|----------|------|
| 跑步 (半马) | 2:10 | 2:00 |

| 力量训练 | 三分化 | 增肌 + 力量提升 |

## 5. 技术选型

| 组件 | 技术选择 | 理由 |
|------|----------|------|
| 运行时 | OpenClaw AgentSkills | 遵循宪法 V3.7 |
| 部署位置 | agents/fitbot/ | 领域智能体 |
| 数据存储 | SQLite + JSON | 轻量、本地存储 |
| 华为数据 | 手动导入 (CSV/JSON) | 无需 API 权限 |
| 飞书集成 | OpenClaw message 工具 | 统一机器人 |
| 定时任务 | cron + OpenClaw | 定时提醒/周报 |

---

## 6. 项目范围

### 6.1 包含 (In Scope)

- 三分化力量训练计划生成
- 跑步训练计划 (半马提升)
- 骑车训练安排
- 高蛋白饮食规划
- 华为手表数据手动导入
- 训练负荷/心率/睡眠分析
- 进度追踪与可视化
- 飞书消息交互 (OpenClaw)
- 定时提醒 (训练/饮食)
- 周报自动生成推送

### 6.2 不包含 (Out of Scope)

- 实时心率监控
- 视频动作指导
- 社交功能
- 电商/商城集成
- 华为 API 直连 (后续迭代)
