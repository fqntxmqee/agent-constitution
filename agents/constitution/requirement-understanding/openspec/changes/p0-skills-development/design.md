## 3. 数据模型

### 3.1 任务数据模型

```typescript
interface Task {
  id: string;
  type: 'development' | 'content' | 'skill' | 'ops' | 'other';
  status: 'pending' | 'running' | 'completed' | 'failed';
  priority: 'P0' | 'P1' | 'P2';
  
  // 分析结果
  analysis?: {
    complexity: 'low' | 'medium' | 'high';
    estimatedSteps: number;
    requiredTools: string[];
    estimatedTime: number; // ms
  };
  
  // 路由信息
  routing?: {
    targetSkill: string;
    reasoning: string;
    factors: Factor[];
  };
  
  // 时间戳
  createdAt: string;
  updatedAt: string;
}
```

### 3.2 技能执行记录

```typescript
interface SkillExecutionRecord {
  taskId: string;
  skillId: string;
  status: 'success' | 'error' | 'pending';
  input: any;
  output: any;
  duration: number; // ms
  timestamp: string;
  error?: string;
}
```
