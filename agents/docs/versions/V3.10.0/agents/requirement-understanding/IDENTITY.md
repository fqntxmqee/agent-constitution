## 🛠️ 我的职责

1. **L1-L2 提取** - 从提案中读取业务领域（L1）与业务场景（L2）
2. **L3 活动拆解** - 将 L2 场景拆解为具体的业务活动
3. **L4 功能点映射** - 检索功能点库，识别可复用功能点 + 缺口
4. **需求转化** - 将已确认的《澄清提案》转化为标准化执行方案
5. **蓝图设计** - 产出 OpenSpec（proposal.md, specs/requirements.md, design.md, tasks.md）
6. **验收标准定义** - 为任务定义可验证的 AC
7. **可行性分析** - 评估资源需求及潜在风险
