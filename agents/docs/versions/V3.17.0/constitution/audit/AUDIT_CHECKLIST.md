## 四、文档引用检查（V3.17.0 新增）

**检查频率**: 每 2 小时定时审计

### A. 新增文档扫描

- [ ] 扫描过去 24 小时新增规范文档
  ```bash
  find agents/constitution -name "*.md" -mtime -1
  ```
- [ ] 检查每个新增文档是否被引用

### B. 引用验证

- [ ] **宪法索引引用**: 检查 CONSTITUTION.md 是否收录
  ```bash
  grep -r "新增文档名" agents/docs/specs/constitution/CONSTITUTION.md
  ```
- [ ] **智能体配置引用**: 检查 AGENTS.md 参考文档章节
  ```bash
  grep -r "新增文档名" agents/constitution/*/AGENTS.md
  ```
- [ ] **任务规约引用**: 检查任务文件引用
  ```bash
  grep -r "新增文档路径" .tasks/
  ```

### C. 违规处理

- [ ] 发现 VIO-008 违规 → 记录到 violation-log.jsonl
- [ ] 通知银河导航员 + 责任智能体
- [ ] 24 小时后验证处理结果
- [ ] 未处理 → 升级（@用户）

### D. 例外情况确认

以下文档无需引用（确认非误报）:
- [ ] 临时报告（审计报告等）
- [ ] 日志文件（*.log, *.jsonl）
- [ ] 任务文件（.tasks/ 目录）
- [ ] 会议纪要
