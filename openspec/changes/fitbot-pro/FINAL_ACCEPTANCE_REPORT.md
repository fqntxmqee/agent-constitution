## 8. 功能演示

### 8.1 运动规划示例

```javascript
const bot = new FitBot();
const plan = bot.generateWorkoutPlan(1);

// 力量训练（哑铃三分化）
- 周一：推 (胸/肩/三头) - 哑铃卧推 4×8-10, 哑铃肩上推举 4×8-10...
- 周三：拉 (背/二头) - 哑铃划船 4×8-10, 哑铃硬拉 4×8-10...
- 周五：腿 (股四头/腘绳/小腿) - 哑铃深蹲 4×8-10...

// 跑步训练（半马配速）
- 周二：轻松跑 45 分钟 @ 6:30-7:00 min/km
- 周四：配速跑 40 分钟 @ 6:20-5:41 min/km (渐进)
- 周六：长距离 90+ 分钟 @ 6:30-7:00 min/km

// 骑车训练（有氧恢复）
- 每日：60 分钟 @ 中低强度，心率 120-140 bpm
```

### 8.2 饮食规划示例

```javascript
const nutrition = bot.generateNutritionPlan(75);

// 高蛋白增肌计划 (75kg 用户)
- 每日热量：2,625 kcal
- 蛋白质：150g (2.0g/kg)
- 碳水：295g
- 脂肪：73g

// 餐单
- 早餐：燕麦 50g, 蛋白 4 个 + 全蛋 2 个，牛奶 250ml, 香蕉 1 根 (35g 蛋白)
- 午餐：鸡胸肉 150g, 米饭 150g, 蔬菜 200g (40g 蛋白)
- 晚餐：鱼肉 150g, 红薯 150g, 蔬菜 200g (40g 蛋白)
- 加餐：蛋白粉 1 勺，坚果 30g, 希腊酸奶 100g (30g 蛋白)
```
