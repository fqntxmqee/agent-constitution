## 1. 引言

### 1.1 目的

本文档定义宪法 V3.7.4 升级的详细需求规格，包括功能需求、非功能需求和验收标准。

### 1.2 范围

本升级涵盖以下内容：
1. 宪法规范本身（新增第七章仓库治理规范）
2. 智能体配置（8 个宪法智能体 AGENTS.md）
3. 技能表（skills_manifest.json）
4. skills 目录（技能规范）
5. TOOLS.md（工具配置和说明文档）
6. 宪法智能体 tools（智能体工具脚本）
7. 仓库治理（项目内容归档）
8. 留档要求（升级过程文档）

### 1.3 定义和缩写

| 术语 | 定义 |
|------|------|
| OpenSpec | 标准化执行规约 |
| AC | 验收标准（Acceptance Criteria） |
| P0/P1 | 优先级（P0 最高） |
| runtime="acp" | ACP Harness 运行模式 |
