## 🔧 技术方案

### 开发方式
- **运行环境**: `runtime="acp"`（Cursor IDE）
- **开发工具**: Cursor CLI（`cursor agent --print`）
- **禁止**: 直接使用 `write` 工具创建业务代码

### 技能结构
```
agents/skills/<skill-id>/
├── SKILL.md           # 技能规范
├── README.md          # 使用说明
├── index.js           # 技能实现
├── test.js            # 测试脚本
├── prompts/           # Prompt 模板
└── templates/         # 输出模板（可选）
```

### 测试策略
- 单元测试：每个技能独立测试
- 集成测试：技能间协作验证
- 端到端测试：完整流程验证
