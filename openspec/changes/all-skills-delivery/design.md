## 3. 技能模块设计

### 3.1 标准结构

```
agents/skills/<skill-id>/
├── SKILL.md           # 技能规范
├── index.js           # 技能实现
├── test.js            # 测试脚本
├── prompts/           # Prompt 模板
├── templates/         # 输出模板
└── README.md          # 使用说明
```

### 3.2 接口标准

```javascript
/**
 * 技能主函数
 * @param {Object} input - 输入数据
 * @returns {Promise<Object>} 输出结果
 */
async function execute(input) {
  return {
    success: true,
    data: {},
    errors: [],
    metadata: {}
  };
}
```
